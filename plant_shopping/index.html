Greenhouse plants

There is a card on this page for each plant in the subset of greenhouse data. Each card has some basic information about the species and links to an incredibly basic html page for the species.

I created this page by writing a python script to iterate over the spreadsheet of greenhouse data and write the code for each plant card. The idea was to then recreate the plant cards using React. I think React is a potential way to do create the plant pages functions but I think there are faster and easier alternatives.

They all have the same photo right now which I would love to change but there are not photos of each greeenhouse plant. The format for getting unique photos would be to name the photo files with the plant id number that I used to create the plant pages and to slightly tweak the phyton to accommodate that.
